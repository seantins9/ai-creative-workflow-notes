# AI Creative Workflow Notes

A small public reference page for testing creative AI workflows before publishing visual assets.

- [ClipTrend.ai](https://cliptrend.ai/) helps turn images and prompts into short AI videos.
- [AIClothSwap](https://aiclothswap.com/) helps preview outfit and clothing changes on a person image.
- [AIChangeHair](https://aichangehair.com/) helps preview hairstyle and haircut ideas from a portrait.

The related GitHub Pages version is published from this repository.
